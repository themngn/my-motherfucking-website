Gibberish text fix

If you see gibberish text in some of the software you use (especially in old one) it could be because of the encoding. Some old software doesn't support unicode. The use old windows single-byte encoding (like windows-1251, windows-1252, etc). Because of it's single-byte nature, it can't represent all the characters in the world. So, if you see gibberish text, you need to specify the encoding manually. You can do it like this:

  1. Open the Region settings from control panel or search for it in the start menu.
  2. Click on the Administrative tab.
  3. Click on the "Change system locale" button.
  4. Set the "Current system locale" to the one that is used by the program.
  5. Optionally, you can check the "Beta: Use Unicode UTF-8 for worldwide language support" checkbox. (This didn't work for me, but it might work for you)
  6. Click OK.
