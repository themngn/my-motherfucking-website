Windows Language Bug

For some unknown reason, Windows 11 has a bug that causes the keyboard layout to appear that can not be removed using the language settings.

It could be remove using Powershell or Registry Editor. But there is a simpler way to do it.

  1. Open the Region settings from control panel or search for it in the start menu.
  2. Click on the Administrative tab.
  3. Click on the "Copy settings" button.
  4. Check the "Welcome screen and system accounts" and "New user accounts" checkboxes.
  5. Click OK.
